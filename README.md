# hc-webhook-plugin

Advanced webhook management and delivery for the Apito API platform.

## Features

- Webhook management
- Event filtering
- Retry mechanisms
- Payload transformation
- Delivery tracking

## Installation

```bash
go build -o hc-webhook-plugin .
```
